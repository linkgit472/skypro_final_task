Git

  • clone - клонирует удаленные репрозиторий из GitHub на компьютер.
  • status - просмотр изменений, которые произошли в репрозитории.
  • add - добавляет файл в отслеживаемые.
  • commit -m - создает запись изменений в репрозитории.
  • push - вносит изменений в удаленный репрозиторий.
